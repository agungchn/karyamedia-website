# Name Tag Custom untuk Event, Kantor, & Wedding

Name tag (tanda pengenal) sering dianggap sepele, padahal berperan penting untuk profesionalisme dan kenyamanan tamu. Berikut panduan memilih name tag custom.

## 1. Fungsi Name Tag
- Identitas peserta seminar/workshop.
- Akses masuk kantor atau pabrik.
- Souvenir pernikahan yang fungsional.

## 2. Pilihan Bahan
- **Akrilik** — elegan, awet.
- **Kayu** — natural, cocok wedding.
- **Logam** — premium untuk eksekutif.
- **PVC/cetak** — ekonomis untuk event besar.

## 3. Desain yang Efektif
Gunakan font yang mudah dibaca, sertakan logo, dan warna brand. Untuk wedding, tambahkan nama mempelai dan tanggal.

## 4. Aksesoris
Klips, pin, atau tali lanyard memudahkan pemakaian. Pilih sesuai suasana acara.
