# Karyamedia Souvenir

Produsen souvenir custom sejak 2001. Berbadan hukum, berbasis Yogyakarta, melayani seluruh Indonesia.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion, CSS keyframes
- **Icons:** Lucide React
- **Fonts:** Geist (sans), Playfair Display (display)

## Project Structure

- `src/app/` — Pages (App Router)
- `src/components/` — UI components
- `src/data/` — Content data (products, categories, articles, testimonials)
- `src/lib/` — Utilities
- `public/images/` — Images organized by product category
- `add.js` — Script to add new blog articles from `article.txt`

## Getting Started

```bash
npm run dev
```

Build:

```bash
npm run build
```
